# arduino-water-tank-full-alarm
사용 부품
- 아두이노 우노 R3 보드
- 비접촉 수위 센서 (XKC-Y25)
- 피에조 능동 부저
- LED
- 워터 펌프 모터
- 푸시 버튼

부품 연결
- 비접촉 수위 센서
  - 5-24V ↔ 아두이노 5V
  - GND ↔ 아두이노 GND
  - OUT ↔ 아두이노 디지털 n번 핀
- 피에조 능동 부저
  - (+) ↔ 아두이노 디지털 2번 핀
  - (-) ↔ 아두이노 GND
- LED
  - (+) ↔ 아두이노 디지털 n번 핀
  - (-) ↔ 아두이노 GND
- 워터 펌프 모터
  - (+) ↔ 아두이노 디지털 n번 핀
  - (-) ↔ 아두이노 GND
- 푸시 버튼
  - 한쪽 핀 ↔ 아두이노 디지털 n번 핀
  - 다른 한쪽 핀 ↔ 아두이노 GND
