# 아두이노를 이용한 제습기 수조 수위 알림 및 버튼으로 작동하는 펌프

## 사용 부품
- 아두이노 우노 R3 보드
- 비접촉 수위 센서 (XKC-Y25)
- 피에조 능동 부저
- LED
- 워터 펌프 모터
- 푸시 버튼

## 부품 연결
- 비접촉 수위 센서
  - 5-24V ↔ 아두이노 5V
  - GND ↔ 아두이노 GND
  - OUT ↔ 아두이노 디지털 5번 핀
- 피에조 능동 부저
  - (+) ↔ 아두이노 디지털 2번 핀
  - (-) ↔ 아두이노 GND
- LED
  - (+) ↔ 아두이노 디지털 13번 핀
  - (-) ↔ 아두이노 GND
- 워터 펌프 모터
  - (+) ↔ 아두이노 디지털 4번 핀
  - (-) ↔ 아두이노 GND
- 푸시 버튼
  - 한쪽 핀 ↔ 아두이노 디지털 3번 핀
  - 다른 한쪽 핀 ↔ 아두이노 GND

## 기능
- 제습기 수조의 수위가 수위 센서의 위치에 다다르면 부저와 LED로 알림
- 버튼을 누르면 제습기 수조 내부의 펌프로 물을 외부로 빼냄
